Ned the Nerd

Modular desktop AI assistant

View on GitHub Get in Touch
PythonPyQt6SpeechRecognition OpenAI APISQLiteModular Architecture

The idea

Most "AI assistant" side projects stop at answering questions in a chat box. Ned is built to go further — a desktop companion that listens through voice, remembers context across a conversation, and can trigger real actions on the machine it's running on, all through a modular, plugin-ready core.

Actively under development

This is an ongoing personal project. Voice recognition, the desktop interface, AI conversation, and the modular code structure are working today; wake-word detection, offline speech recognition, and local LLM support are in progress or planned.

Screenshots

What's working

  • Voice interaction: speech-to-text recognition with a low-latency listening architecture and context-aware responses.
  • Conversational AI: an OpenAI-backed core that holds conversation context rather than treating each message in isolation.
  • Productivity automation: launches applications and executes desktop commands through a plugin-ready architecture built for future extensions.
  • Persistent memory: SQLite-backed storage so context can survive between sessions.
  • Modern desktop UI: built on PyQt6 for a native, responsive interface rather than a bare terminal loop.

What's next

  • Wake-word detection so Ned can be triggered hands-free.
  • Offline speech recognition via Faster Whisper or Vosk, reducing dependence on cloud APIs.
  • Local LLM support for a fully offline conversational mode.
  • Calendar integration, smart reminders, and workflow automation to push Ned further into "productivity tool" territory.

Architecture

Ned is split into clearly separated modules — ai/ for the conversational backend, core/ for orchestration, interfaces/ for the PyQt6 UI, database/ for persistence, and a dedicated wakeword/ module for the in-progress always-listening feature. The intent is a codebase where new capabilities can be added as plugins rather than bolted onto a single script.
