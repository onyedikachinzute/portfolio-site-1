Kachi's To-Do List

Minimalist Flask-powered task manager

View on GitHub Get in Touch

The idea

A clean, responsive to-do list app built to be genuinely pleasant to use day-to-day — not just a CRUD exercise. Tasks can be added, marked complete, and deleted, all backed by a lightweight Flask server with zero database setup required.

Screenshots

Features

  • Add, mark, and delete tasks with smooth micro-animations on every action.
  • Data persistence via a lightweight local JSON file — no database setup needed.
  • Dark / light mode toggle with the preference saved in localStorage.
  • Fully responsive layout that works cleanly on both desktop and mobile.

How it works

When a task is added, it's written to tasks.json. Toggling a task's completion state fires an AJAX POST call to the Flask backend, which updates the done status. Deleting a task fades it out client-side before removing it from the JSON store. The Flask server (app.py) handles all of this through simple REST-style routes, with the frontend template in templates/index.html and styling/logic split across static/style.css and static/script.js.

# running it locally
git clone https://github.com/onyedikachinzute/to-do_list.git
cd to-do_list
pip install flask
python app.py
# then visit http://127.0.0.1:5000/
