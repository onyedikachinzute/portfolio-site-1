06 / About

Who I am, how I work

What drives me, and how I approach building things.

I'm Onyedikachi "Kachi" Nzute, a Computer Science student and self-taught developer who likes turning ideas into software that actually gets used. I specialize in Python-based systems — desktop apps, computer vision pipelines, and backend logic — with a growing focus on the kind of clean, dependable engineering that holds up outside a tutorial.

I'm currently working through a personal 50-project roadmap, a self-imposed challenge to explore different technologies and problem spaces through practical builds — from REST APIs and databases to computer vision and automation scripts. Each project is picked to sharpen a specific skill.

My philosophy: keep learning, keep building, keep improving. I don't shy away from complex problems — every bug and refactor is part of the process. I document what I learn and regularly revisit old code to see how far I've come.

Technical focus: Python is my main tool, alongside HTML, CSS, and JavaScript on the web side, and PostgreSQL/SQLite for data. I'm especially interested in computer vision, API design, and building software that stays useful outside ideal conditions — like the offline-first constraint behind the OES ID Extractor.

Mindset & work ethic: I value consistency, clarity, and reliability. Whether I'm debugging a detection pipeline or wiring up an admin dashboard, I try to write code that's easy for someone else — or future me — to pick up and understand.

Beyond code: I'm into tech communities, UI design trends, and lifelong learning. My goal isn't just to write code — it's to build things that make someone's actual workflow easier.
